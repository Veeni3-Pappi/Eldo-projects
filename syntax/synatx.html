CSS- Cascading style sheets

Css is a stylesheet language used to describe the presentation of a document written in HTML or XML
. It allows you to control the layout and appearance of web pages.

CSS Syntax 

   selector{
    property: value;
    property: value;
}

In the above sytax

  • Selector: Specifies the HTML element(s) to which the styles should be applied.
  • Property: Contains the CSS properties and their corresponding values.
  • Value: Specifies the actual value for each CSS property.

Inline css

you can apply Css directly to an HTML element using the `style` attribute.
